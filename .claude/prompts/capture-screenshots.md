# Capture Screenshots

Navigate through the application and capture screenshots for every major user-facing page.

Follow the rules in:

- screenshots.md
- playwright.md
- login-workflow.md

For each page:

1. Wait until loading has completed.
2. Capture the entire application window.
3. Save the image using the required naming convention.

Store all screenshots in:

docs/screenshots/

Do not generate documentation during this task.
