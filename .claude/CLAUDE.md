# Recruitment AI — Project Guide

## Project Overview

Recruitment AI is an AI-powered recruitment platform built with:

- Backend: FastAPI
- Frontend: React + TypeScript
- Build Tool: Vite
- Styling: Tailwind CSS

The platform manages:

- Authentication
- Job Management
- Candidate Applications
- Resume Parsing
- AI Candidate Scoring
- Interview Management
- Reports
- User Management
