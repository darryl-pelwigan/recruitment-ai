# User Manual Standards

## Audience

Primary audience:

Non-technical users.

Assume the reader has never used the application before.

---

# Document Structure

Generate the manual using this order unless instructed otherwise.

1. Cover Page

2. Document Information

- Version
- Date
- System Name

3. Table of Contents

4. Introduction

5. System Requirements

6. Creating an Account and Logging In

7. Dashboard

8. Browsing and Posting Jobs

9. Applying to Jobs and Tracking Applications

10. Managing Applicants and the Pipeline

11. Saved Jobs and Saved Applicants

12. Resumes

13. Profile

14. User Management (Administrators Only)

15. Frequently Asked Questions

16. Troubleshooting

17. Logging Out

---

# Role-Based Sections

Recruitment AI has three roles: `applicant`, `recruiter`, and `admin`.

Some pages only apply to certain roles. Label these sections clearly, for example:

"**Posting a Job** (Recruiters only)"

"**User Management** (Administrators only)"

Do not document role-restricted pages as if every reader can access them.

---

# Each Section

Every section should include:

## Overview

Explain the purpose of the page.

---

## Screenshot

Insert a relevant screenshot.

---

## Procedure

Provide numbered instructions.

---

## Notes

Highlight important information.

---

## Tips

Provide optional best practices when helpful.

---

# Writing Rules

Always:

- Explain why the user performs an action.
- Use short sentences.
- Use consistent terminology.
- Keep instructions sequential.

Never assume prior knowledge.

---

# Screenshots

Insert screenshots immediately after introducing a page.

Reference screenshots naturally.

Example:

"The Dashboard is shown below."

---

# Formatting

Use:

Heading 1

Heading 2

Heading 3

Bullet lists

Numbered procedures

Tables where appropriate

---

# Output

Preferred output:

Microsoft Word (.docx)

Location:

docs/generated/

Filename:

UserManual.docx

---

# Completion Checklist

Before finishing:

- All screenshots included
- All chapters completed
- Table of Contents updated
- Grammar checked
- No placeholder text remains
- Instructions verified against the application
